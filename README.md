# Iris Dataset Clustering

This project applies unsupervised machine learning techniques to the Iris dataset using Python. The species column is removed because the goal is to group flowers based only on their numerical features.

## Algorithms Used

- KMeans Clustering
- Hierarchical Clustering

## Objective

The objective of this project is to understand and apply clustering techniques to a real-world dataset. The clusters are visualized using petal length and petal width to observe how the data points are grouped.
